Rkz Platform

PROJECT DESCRIPTION

Is a platform that analyzes student activities in online classes using artificial intelligence through webcams and microphones that allow the teacher to easily control students and measure their performance. User state-of-the-art technologies such as eye trackers, CNN models and signal processing. This project was top 20 project in Homathon
